# 天线的解耦技术

# 1. 天线耦合的产生

​	天线通常不是作为单个单元来应用，且微带天线的单元增益并不高，常常需要将单元天线组合成天线阵列来应用于各个场景。天线阵列由大量天线单元组成，天线阵元之间的距离越大，其体积就越大并且其栅瓣情况会严重，单元间距受到限制，但距离越小互耦越严重，所以为实现小型化的需求，减少天线阵元之间的距离，这些天线单元排布在紧凑的空间内。在这种情形下，天线单元之间，会随着相位配置的不同，单元的电磁场会波及到相邻的单元，使得单元间的耦合显著增加。随着科技的快速发展，天线单元的数量和密度比以往任何时候都要高，对于阵列天线，较高的互耦会严重影响天线的各项性能指标，比如恶化阻抗匹配、降低辐射效率以及引起辐射方向图畸变等，使得天线互耦成为阵列设计中的一个严重问题。综上所述，在频谱资源变得越来越宝贵的当下，在有限的空间内放置天线元件的同时保持高隔离度和天线性能是一项挑战，因此，简单有效的解耦受到了广泛的关注。如何在有限的空间下实现宽带毫米波天线之间的高隔离，已经成为天线研究领域亟待解决的重要问题[1]。

# 2. 解耦方式研究

## 2.1 微带天线

​	贴片天线的窄带特性与其高 Q 因子参数密切相关,追求较宽的阻抗带宽一直是贴片天线的设计标准之一,过去几十年来,这方面取得了重大进展。

### 2.1.1 微带天线扩展带宽

​	第一类是引入周期性结构来扩大天线的辐射孔径。较大的辐射孔径可导致较宽的阻抗带宽。然而,对于某些应用,可能没有足够的空间来容纳大规模的周期结构。

​	第二类侧重于在天线的工作频带内产生额外的谐振。已有使用 L 形探针馈电,或不同形状馈电微带贴片天线、缝隙微带贴片天线、U 形缝隙、V 形缝隙、环形缝隙和堆叠贴片天线，或者激发贴片天线的不同模式[3]等技术来实现此目的。
