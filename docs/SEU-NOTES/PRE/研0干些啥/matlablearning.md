### 保存和加载变量save&load

如果想要将工作区中的所有变量保存到.mat文件中，直接使用代码

```matlab
save filename
```

如果想单独保存某个变量，使用

```matlab
save filename vername
```

如果想加载.mat文件，使用```load```函数

```matlab
load filename
```

如果想加载某个变量，使用

```matlab
load filename vername
```
